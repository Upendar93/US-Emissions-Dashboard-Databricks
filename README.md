# United States Emissions Dashboard (Databricks)

## Project Overview
This project analyzes **United States emissions data (EPA 2023)** and builds an interactive dashboard using **Databricks** to visualize emission patterns across counties and states.

The dashboard provides insights into geographic emission distribution, population relationships, and the counties contributing the most emissions.

---

## Dashboard Features

### 1. Emissions Per Location
Geospatial map visualization showing emissions across the United States.

### 2. Emissions vs Population
Scatter plot analyzing the relationship between population size and emissions per person.

### 3. Emissions Percentage by State
Donut chart showing emission contributions from the top states.

### 4. Top 10 Counties by Emissions
Bar chart identifying counties with the highest emissions.

## Dataset

Source: **Environmental Protection Agency (EPA)**  
Year: **2023**

The dataset includes:

- State
- County
- Population
- Total Emissions
- Emissions Per Person
- Latitude
- Longitude

---

## Key Insights

- A small number of states contribute to a significant portion of emissions.
- Population-dense areas tend to have lower emissions per person.
- Certain counties dominate total emissions due to industrial or economic activity.
